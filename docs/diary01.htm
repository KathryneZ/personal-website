NM2207 Project Diary Entry 01

My final project is a combined spending and mood tracker that allows users to input income or expenses by category and log their daily mood using an interactive calendar. The aim is to create a visually clean interface that encourages reflection on financial habits and emotional patterns.

This week, I focused on building the overall layout/framework using HTML and CSS, following the planned layout I had drawn out (see below).

I implemented two main buttons for adding expenses and income, styled responsively with colour differentiation (red and green). When clicked, each button triggers a pop-up form that allows users to input transaction details (amount, category, description, date). I also structured the main layout into two columns: the left for categorised financial records, and the right for a mood tracker. The calendar displays 30 days, and I plan to generate this with JavaScript, with each cell clickable to prep for future mood-tagging functions.
